# 폭탄 피하기 게임

## 소개
오직 html, css, javascript(es6 이상 기능도 사용)만을 사용하여 구현한 간단한 게임


### 실행하기

```
 1. Clone or download 하기
 2. console를 통해 해당 폴더로 이동
 3. npm install 실행
 4. npm start
 5. 웹 브라우저에 http://localhost:5000 실행
```

### 특징
```
 + 해당 repository(https://github.com/meth-meth-method/super-mario)에 있는 코드를 사용하여 구현했다.
 + html과 css을 사용하되 디자인은 최소화했다.
 + localStorage에 기록을 저장했다.
 + spriteSheet를 사용하여 최대한 애니메이션을 구현했다.
```
